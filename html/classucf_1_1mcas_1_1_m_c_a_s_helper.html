This function is called to remove a descriptor which is in conflict with a MCAS operation.

If the descriptor is a MCH, then a check is performed to determine if it is for the same operation. If so, then it is treated differently. Otherwise, the descriptors complete function is called.

This function is called after this objects rc count was incremented.

It acquires a temporary HP watch on the MCAS op (via last_row_), ensures it is is associated, and if so returns true. If it is not associated or it was removed, it returns false
