Detailed Description

This class represents the progress assurance scheme employed by our library.

For this scheme to be effective, each operation which may indefinetly prevent the progress of some other operation must call the static function offer_help This ensures that if a thread is continually failing its operation, then after a finite number of tries all thread will be helping. The number of failures is MAX_FAILURES + (number of threads^2)* help_delay
